# Halalisani Ngema — Professional Portfolio

A professional landing page / portfolio website for Halalisani Ngema, built with pure HTML, CSS, and vanilla JavaScript.

## 🌐 Live Site

Hosted via GitHub Pages: [https://halalisanin.github.io](https://halalisanin.github.io)

## ✨ Features

- Fully responsive design (mobile-first)
- Dark gold editorial aesthetic
- Smooth scroll animations with Intersection Observer
- Sticky navigation with scroll effects
- Animated hero section with geometric decorations
- Interactive skill chips with hover effects
- Scroll-triggered timeline animations
- Certificate showcase cards
- Contact section with live links
- Zero dependencies — pure HTML/CSS/JS

## 📁 Structure

```
halalisani-portfolio/
└── index.html    # Single-file portfolio site
```

## 🚀 Deploy to GitHub Pages

1. Go to your repo on GitHub
2. Settings → Pages
3. Source: Deploy from branch → `main` → `/ (root)`
4. Save — your site will be live at `https://halalisanin.github.io`
